
```
1.  u：撤销上一次的操作。
2.  Ctrl + r：重做上一次撤销的操作。
3.  a：在光标后插入文本。
4.  A：在行末插入文本。
5.  o：在当前行下方插入新的一行。
6.  O：在当前行上方插入新的一行。
7.  x：删除光标所在处的字符。
8.  dd：删除当前行。
9.  dw：删除光标所在处到单词末尾的字符。
10.  d$：删除光标所在处到行末的字符。
11.  r：替换光标所在处的字符。
12.  s：删除光标所在处的字符并进入插入模式。
13. :e 文件路径：打开另一个文件。
14.  :sp 文件路径：在一个新的窗口中水平分屏打开另一个文件。
15.  :vsp 文件路径：在一个新的窗口中垂直分屏打开另一个文件。
16. ```

```
1.  :wq!：强制保存并退出当前打开的文件。
    
2.  :q!：强制退出当前打开的文件并放弃未保存的修改。
    
3.  :set hlsearch：开启搜索高亮。
    
4.  :set nohlsearch：关闭搜索高亮。
    
5.  :set ignorecase：忽略大小写进行查找。
    
6.  :set noignorecase：区分大小写进行查找。
    
7.  :set list：显示文本中的制表符、行末空格等不可见字符。
    
8.  :set nolist：关闭不可见字符的显示。
    
9.  :set wrap：自动换行。
    
10.  :set nowrap：不自动换行。
    
11.  :set expandtab：将制表符自动转换为空格。
    
12.  :set noexpandtab：不将制表符自动转换为空格。
    
13.  :set tabstop=4：设置制表符宽度为4个空格。
    
14.  :set number：显示行号。
    
15.  :set nonumber：不显示行号。
    
16.  :set mouse=a：开启鼠标支持。
    
17.  :set clipboard=unnamedplus：设置系统剪贴板为vim剪贴板。
    
18.  ggVG：选中整个文件。
    
19.  :%s/old/new/g：将文件中所有的old替换成new。
    
20.  :%s/old/new/gc：将文件中所有的old替换成new并进行确认。
    
21.  :r file.txt：将另一个文件file.txt的内容插入到当前文件的光标位置。
    
22.  :.!date：在当前文件中插入当前日期。
    
23.  :!cmd：执行命令cmd，例如：!ls列出当前目录下的文件。
    
24.  :w FILENAME：将当前文件另存为FILENAME。
    
25.  :e!：放弃所有未保存的修改并重新打开当前文件。
    
26.  :set spell：开启拼写检查。
    
27.  :set nospell：关闭拼写检查。
    
28.  :setlocal spell：对当前文件开启拼写检查。
    
29.  :set spelllang=en_us：设置拼写检查的语言为英语。
    
30.  Ctrl + w + w：在窗口之间进行切换。
    
31.  Ctrl + w + q：关闭当前窗口。
    
32.  Ctrl + w + s：水平分割窗口。
    
33.  Ctrl + w + v：垂直分割窗口。
    
34.  Ctrl + w + =：平均调整所有窗口的大小。
    
35.  Ctrl + g：显示文件名、文件状态、行号等信息。
    
36.  Ctrl + o：返回上一个位置。
    
37.  Ctrl + i：返回下一个位置。
    
38.  Ctrl + x + Ctrl + f：自动补全文件路径。
    
39.  Ctrl + x + Ctrl + o：自动补全命令。
    
40.  Ctrl + a：将光标所在的数字加1。
    
41.  Ctrl + x：将光标所在的数字减1。
    
42.  :reg：查看寄存器内容。
