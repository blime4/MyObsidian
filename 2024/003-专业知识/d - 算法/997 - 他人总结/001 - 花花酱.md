# a - 花花酱

+ 要刷多少题：
	+ 每个类型10-20题 动态规划 多多益善
	+ 总共 200 - 300 题
+ 如何刷题：
	+ 同类型的题目一起刷：
		+ eg: 周一：链表， 周二：树， 周三:搜索， 周四：动态规划
	+ 第一遍：5分钟想不出就看答案
	+ 第二遍：尝试不看答案完整实现一遍
	+ 第三遍：尝试快速实现，10-20分钟实现不了就看答案
+ 看代码很重要！看代码很重要！看代码很重要！
	+ 看至少3-5种不同实现，分析优劣，分析快慢
	+ 学习新的语言/算法/数据结构/API/模板/最佳实践
+ 刷题过程中培养的能力
	+ 扩宽思路，至少能看出用什么样的算法
	+ 数据规模-》时间复杂度推算
	+ 代码风格：
		1. 一致性：命名/缩进/括号/换行
		2. 有意义的变量名
