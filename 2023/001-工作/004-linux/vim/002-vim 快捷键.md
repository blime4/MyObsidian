
```
1.  u：撤销上一次的操作。
2.  Ctrl + r：重做上一次撤销的操作。
3.  a：在光标后插入文本。
4.  A：在行末插入文本。
5.  o：在当前行下方插入新的一行。
6.  O：在当前行上方插入新的一行。
7.  x：删除光标所在处的字符。
8.  dd：删除当前行。
9.  dw：删除光标所在处到单词末尾的字符。
10.  d$：删除光标所在处到行末的字符。
11.  r：替换光标所在处的字符。
12.  s：删除光标所在处的字符并进入插入模式。
13. :e 文件路径：打开另一个文件。
14.  :sp 文件路径：在一个新的窗口中水平分屏打开另一个文件。
15.  :vsp 文件路径：在一个新的窗口中垂直分屏打开另一个文件。
