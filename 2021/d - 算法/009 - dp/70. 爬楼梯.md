这题很像斐波那契数列：爬到第 `n` 级台阶的方法个数等于爬到 `n - 1` 的方法个数和爬到 `n - 2` 的方法个数之和。

```c++
class Solution {
public:
    int climbStairs(int n) {
        if(n < 2) return n;
        int p1 = 1, p2 = 1 ,ans = 0;
        for(int i=2;i<=n;i++){
            ans = p1 + p2;
            p1 = p2;
            p2 = ans;
        }
        return ans;
    }
};
```
