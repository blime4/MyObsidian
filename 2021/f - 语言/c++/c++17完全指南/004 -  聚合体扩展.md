# 004 - 聚合体扩展


自从 C++17 起，聚合体可以拥有基类。
```c++
struct Data {  
	std::string name;  
	double value;  
};

struct MoreData : Data {  
	bool done;
}

MoreData y{{"test1", 6.778}, false};
// or

MoreData y{"test1", 6.778, false};
```
